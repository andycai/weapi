# Weapi

Weapi is the lightweight open-source headless CMS wrote in Golang.

Golang + Go Fiber + Gorm + Alpine.js

Modules include: 

- Site
- Content
    - Category
    - Post
    - Page
    - Media
- Setting
    - Group
    - GroupMember
    - User
    - Config
- Activity
    - Activity
    - Club
    - ClubMember
